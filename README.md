GET THE AUR HERE:https://aur.archlinux.org/packages/viewery OR yay -S viewery

ViewerY is a lightweight terminal system information tool for Linux.

It displays basic hardware and system information in a clean ASCII layout.

And this project is open source.

Features

- CPU information
- GPU detection (NVIDIA + fallback support)
- RAM usage
- Motherboard info
- Kernel & OS info
- Clean terminal layout
- Lightweight and fast
- and system info
- workes with ANY distro


!RUN THIS WITH   "python viewery.py" AND MAKE SURE YOU HAVE PYTHON INSTALLED!
